===
Название: htop
Описание: Постраничный просмотр текстовых файлов
Синтаксис: htop
Пример: htop
Примечание:  лучше top
===
Название: more
Описание: Постраничный просмотр текстовых файлов
Синтаксис: more [опции] [файл]
Пример: more document.txt
Примечание: Позволяет пролистывать файл страница за страницей с помощью пробела

===
Название: head
Описание: Вывод начала файла
Синтаксис: head [опции] [файл]
Пример: head -n 20 file.log
Примечание: По умолчанию показывает первые 10 строк, можно изменить количество флагом -n

===
Название: tail
Описание: Вывод конца файла
Синтаксис: tail [опции] [файл]
Пример: tail -f /var/log/syslog
Примечание: Флаг -f позволяет следить за обновлениями файла в реальном времени

===
Название: unzip
Описание: Распаковка ZIP-архивов
Синтаксис: unzip [опции] архив.zip
Пример: unzip -d target_dir archive.zip
Примечание: Флаг -d позволяет указать директорию для распаковки

===
Название: grep
Описание: Поиск текста по шаблону
Синтаксис: grep [опции] шаблон [файлы]
Пример: grep -r "error" /var/log/
Примечание: Флаг -r включает рекурсивный поиск по директориям, -i игнорирует регистр

===
Название: df
Описание: Отображение информации о дисковом пространстве
Синтаксис: df [опции]
Пример: df -h --total
Примечание: Флаг -h показывает размеры в человеко-читаемом формате, --total добавляет итоговую строку

===
Название: top
Описание: Динамический просмотр запущенных процессов
Синтаксис: top [опции]
Пример: top -u username
Примечание: Покажает процессы в реальном времени, можно сортировать по разным параметрам

===
Название: ps
Описание: Статический снимок запущенных процессов
Синтаксис: ps [опции]
Пример: ps aux --sort=-%mem
Примечание: aux показывает все процессы системы, можно сортировать по потреблению памяти или CPU

===
Название: zip
Описание: Создание ZIP-архивов
Синтаксис: zip [опции] архив.zip файлы
Пример: zip -r project.zip src/ docs/
Примечание: Флаг -r включает рекурсивное архивирование директорий

===
Название: find
Описание: Поиск файлов и директорий
Синтаксис: find [путь] [опции] [выражение]
Пример: find /home -name "*.txt" -mtime -7
Примечание: Можно искать по имени, размеру, времени изменения и другим атрибутам
